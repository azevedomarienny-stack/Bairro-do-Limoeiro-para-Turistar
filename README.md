# Bairro do Limoeiro para Turistar

Projeto desenvolvido com HTML, CSS e JavaScript que simula um portal turístico temático inspirado no universo do Bairro do Limoeiro.

## Objetivo

Aplicar conceitos de desenvolvimento front-end, responsividade, integração com APIs e manipulação do DOM por meio da construção de uma aplicação web interativa.

## Funcionalidades

* Exibição de clima em tempo real utilizando a API OpenWeatherMap
* Layout responsivo com Bootstrap 5
* Carrossel de atrações turísticas
* Seção de notícias e eventos
* Simulação de compra de produtos
* Simulação de passagem turística
* Sistema de avaliações com validação de formulário
* Modais interativos
* Navegação com rolagem suave

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Bootstrap 5
* Bootstrap Icons
* OpenWeatherMap API
